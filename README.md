# AE Medicals – Website

Modern, animated marketing site for **After Effects Medicals LLC**.

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
# Python
python3 -m http.server 8080

# Node (npx)
npx serve .

# Then open http://localhost:8080
```

## Stack

- **HTML5** – semantic sections, accessibility
- **CSS3** – custom properties, grid, keyframe animations, responsive
- **GSAP 3 + ScrollTrigger** – hero timeline, scroll reveals, count-up stats, parallax

## Features

- Hero with gradient orbs and grid background
- Scroll-triggered reveals and staggered strength cards
- Animated stat counters (500+, 10+, 300+, 82M)
- Magnetic-style hover on buttons and nav
- Parallax orbs on scroll
- Mobile menu with animated open/close
- Respects `prefers-reduced-motion`
